# Proyecto Final - Juego

## Descripción
Este proyecto consiste en el desarrollo de un videojuego como proyecto final. El juego permite al usuario interactuar con diferentes elementos, cumplir objetivos y avanzar dentro del sistema mediante mecánicas básicas de juego.

## Motivación
La motivación de este proyecto es aplicar los conocimientos aprendidos durante el curso en un proyecto práctico y creativo, desarrollando un juego que sea funcional, entretenido y que permita practicar lógica de programación y diseño.
